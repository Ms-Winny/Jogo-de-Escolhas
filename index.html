Colinhas de HTML

A tag h2, forma o subtítulo de uma página, ele é menor que o h1 e maior que os outros hs, inclusive é a que estamos usando aqui.

Através da tag h1 você pode criar títulos de uma página, por boa prática o ideal é a página ter apenas um h1, estamos utilizando dois h1 nessa página, o primeiro é o "Colinhas de HTML" e o segundo está logo abaixo desse texto. Mas se os dois são h1 porque estão diferentes? Basicamente eu inseri um código em CSS que manda o texto do h1 de cima ter o tamanho da fonte um pouco maior, isso é chamado de estilização Inline. Pra finalizar, esse texto aqui foi escrito usando h3
